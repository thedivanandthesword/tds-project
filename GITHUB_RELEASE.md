# TDS v1.0.1 — First Official Release

This is the first official release of **The Divan and the Sword (TDS)**, an independent scholarly research project dedicated to the study of Iranian political tradition, bureaucracy, governance, and the Persianate world.

This release establishes the project's public documentation, website, governance framework, and citation metadata. It provides a stable and citable baseline for future development and scholarly publication.

## Highlights

- Initial official public release of the TDS project.
- Finalized core project documentation.
- Improved repository organization and navigation.
- Published the project website using GitHub Pages.
- Added release documentation and publication workflow.
- Added project governance and versioning documentation.
- Added GitHub Pages deployment documentation.
- Added a custom 404 page for improved website navigation.
- Prepared the repository for Zenodo archival and DOI registration.
- Added temporary citation guidance until a DOI is assigned.
