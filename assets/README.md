# Assets

This directory contains static resources used throughout The Divan and the Sword (TDS) website and project repository.

## Purpose

The `assets/` directory stores static resources used throughout the project's GitHub Pages website and documentation, including images, logos, icons, stylesheets, and other supporting files.

## Directory Structure

The directory may contain the following subdirectories:

- `images/` — Project logos, figures, and website images
- `css/` — Cascading Style Sheets (CSS)
- `js/` — JavaScript files (if applicable)
- `icons/` — Icons and favicons
- `downloads/` — Downloadable project resources (if applicable)

Additional subdirectories may be added as the project evolves.

## Usage

Assets should be organized consistently and referenced using relative paths whenever possible to ensure compatibility with GitHub Pages and local development environments.

Static resources should support the project's documentation and website while preserving the integrity, reproducibility, and long-term stability of published releases.
