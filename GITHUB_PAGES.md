# GitHub Pages

The Divan and the Sword (TDS) is published using **GitHub Pages**.

The project website provides access to the project's documentation, governance framework, publication information, and other public resources.

## Website

https://thedivanandthesword.github.io/tds-project/

## Repository

https://github.com/thedivanandthesword/tds-project

## Deployment

The website is automatically deployed from the project's GitHub repository using GitHub Pages.

Before each release, verify that:

- GitHub Pages is active.
- The latest deployment completed successfully.
- The repository is in a clean state (`git status`).

The project includes a custom 404 page to improve navigation when users encounter broken or outdated links.

## Build Status

The latest deployment status can be viewed in the **Actions** tab of the repository.
