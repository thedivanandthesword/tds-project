
# The Divan and the Sword  
  
**The Divan and the Sword (TDS)** is an evidence-based historical research project exploring the intersection of narrative, power, and documentation in classical and modern contexts.

## About  
This repository contains the documentation, metadata, and research outputs of the TDS project. The project adheres to strict principles of traceability, objective evidence, and milestone-based review.
